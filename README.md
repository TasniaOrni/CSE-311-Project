# CSE-311-Project

The main purpose of this project to introduce ourselves to the concepts of Relational Database Systems.

## Technologies used:

- Node.js
- Express.js
- MySQL
- EJS

## Getting Started

### Development Environment

- node.js 16.X
- npm 6.X

### Installing & Configuration

0. Install yarn globally

```
    npm install yarn -g
```

1. Install dependencies

```
    yarn
```

2. Run the server

```
    yarn run dev
```
